## Описание класса Service 
Класс для работы с услугами химчистки.

### Атрибуты 
+ ID : Integer
+ NameOfTheService : String
+ Difficulty : String
+ Price : Decimal

### Описание атрибутов
+ ID : Int - идентификатор в БД
+ NameOfTheService : String - наименование услуги
+ Difficulty : String - сложность исполнения услуги
+ Price : Decimal - цена
