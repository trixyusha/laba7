## Описание класса Client 
Класс для работы с клиентами химчистки.

### Атрибуты 
+ ID : Integer
+ Name : String
+ Phone : String
+ Sale : Integer

### Описание атрибутов
+ ID : Int - идентификатор в БД
+ Name : String - ФИО клиента
+ Phone : String - номер телефона клиента
+ Sale : Integer -  скидка
